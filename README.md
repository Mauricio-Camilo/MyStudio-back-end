## How to run
1. Clone this repository
2. Install dependencies
```bash
npm i
```
3. Create a .env file with the following fields:
```bash
PORT=
DATABASE_URL=
SECRET=
EXPIRES_IN=
NODE_ENV=
```
4. Run the back-end with
```bash
npm run dev
```
